Telerik.Sitefinity.Samples.Education
====================================

Telerik International University (TIU) is a sample university website, which demonstrates how you can build an interactive and engaging university website using Sitefinity's default functionality. The development of the website is accomplished entirely through Sitefinity's web-based UI. This website is being created by 1 designer and 1 developer and it only took 1 week which is a testament to the productivity gains offered by Sitefinity.
